# Daily_Diary
### 개인프로젝트
### 내 손안에 데일리 다이어리 (https://jay-dailydiary.web.app/)

<br>

📓 제작기간 

23.02.26 ~ 23.03.05

<br>

📓 사용기술 

React.js , CSS, Local Storage, Firebase

<br>

### 구현계획
1. 로딩화면
2. 일기 리스트  
3. 새로운 일기 추가하기
4. 추가한 일기 상세페이지
5. 일기 수정하기
6. 일기 삭제하기
7. 달마다 일기 필터링
8. 최신순, 오래된순 필터링
9. 감정별 필터링
